# Page Summarizer — AI-агент «Краткое резюме страницы»

## Цель проекта

AI-агент, который загружает веб-страницу по URL и формирует краткое резюме
содержания (3-5 предложений) с помощью GPT-4o.

## Демо

**Рабочий проект:** [http://83.222.16.42:5001](http://83.222.16.42:5001)

## Логика работы

1. Пользователь вводит URL в веб-форму
2. Агент загружает HTML-страницу и извлекает чистый текст
3. Текст отправляется в GPT-4o с промптом: «Сформулируй суть текста в 3-5 предложениях»
4. Резюме отображается в веб-интерфейсе

## Структура проекта

page_summarizer/ ├── app.py # Flask-приложение (веб-интерфейс) ├── openai_module.py # Логика агента (загрузка страницы + резюме) ├── openai_template.py # Модуль работы с OpenAI API (polza.ai) ├── templates/ │ └── index.html # Фронтенд ├── requirements.txt # Зависимости ├── Dockerfile # Docker-образ └── .env # API-ключ (не в репозитории)


## Функциональные требования

- Вход: URL веб-страницы
- Выход: строка с 3-5 предложениями
- Модель: GPT-4o (через Polza.ai API)
- Повторы: до 3 попыток с линейным back-off
- Логирование: уровни INFO / WARNING / ERROR

## Нефункциональные требования

- Python 3.12
- Типизация (аннотации типов)
- Безопасное хранение ключей (.env)
- Деплой через Docker на VPS

Приложение будет доступно на http://localhost:5001

Технологии
